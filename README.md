# RN Challenge

Simple MetaWeather client using react-native.

## Development

Install dependencies:

```bash
$ yarn
```

Run it:
```bash
$ yarn start
```

## Technologies used

* React Native
* Typescript
* Eslint
* Prettier

## Code Style

The Airbnb JavaScript code style is used in this project.

To run a linting tool (eslint) to check the code style, run:

```bash
$ yarn lint
```
